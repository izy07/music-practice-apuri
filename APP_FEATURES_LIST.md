# アプリ機能一覧

## 📱 メインタブ（下部タブバーに表示）

### 1. カレンダー（index.tsx）
- **機能**: 練習記録のカレンダー表示
- **詳細**:
  - 月次カレンダー表示
  - 日別の練習時間記録
  - クイック記録（音声入力対応）
  - 練習記録の追加・編集・削除
  - 録音ファイルの確認
  - イベント・スケジュール表示
  - 目標のカレンダー表示（show_on_calendar対応）
- **状態**: ✅ 有効

### 2. タイマー（timer.tsx）
- **機能**: 練習時間の計測
- **詳細**:
  - タイマーモード
  - ストップウォッチモード
  - カスタム時間設定
  - クイック設定
  - 練習完了時のカレンダー記録オプション
  - 音量・バイブレーション設定（非表示）
- **状態**: ✅ 有効

### 3. 目標（goals.tsx）
- **機能**: 目標の設定と管理
- **詳細**:
  - 個人目標（短期・長期）
  - 目標の進捗管理
  - 完了目標の表示
  - カレンダー連携（show_on_calendar）
- **状態**: ✅ 有効

### 4. チューナー（tuner.tsx）
- **機能**: 楽器のチューニング
- **詳細**:
  - リアルタイム音程検出
  - 楽器別チューニング設定
  - 基準音（A4）の周波数調整
  - 視覚的インジケーター
  - 音名表示（日本語・英語）
  - セント表示
- **状態**: ✅ 有効

### 5. 共有（share.tsx）
- **機能**: 組織・グループ管理
- **詳細**:
  - 組織の作成（ソロモード対応）
  - 組織への参加（パスワード認証）
  - 組織一覧表示
  - 組織ダッシュボードへの遷移
  - 組織設定への遷移
- **状態**: ✅ 有効

### 6. 設定（settings.tsx）
- **機能**: アプリ設定の管理
- **詳細**:
  - プロフィール設定への遷移
  - マイライブラリへの遷移
  - 録音ライブラリへの遷移
  - 主要機能設定への遷移
  - 通知設定への遷移
  - 使い方ガイドへの遷移
  - 言語設定への遷移
  - 法的情報への遷移
  - プライバシー設定への遷移
  - アプリ共有機能への遷移
  - ログアウト
- **状態**: ✅ 有効

## 🔧 その他の機能

### コンポーネント
- **QuickRecordModal**: クイック記録モーダル（音声入力対応）
- **PracticeRecordModal**: 練習記録モーダル
- **EventModal**: イベントモーダル
- **PracticeStatsModal**: 練習統計モーダル
- **PostureCameraModal**: 姿勢カメラモーダル
- **AudioRecorder**: 音声録音コンポーネント
- **TuningSoundPlayer**: チューニング音再生
- **EventCalendar**: イベントカレンダー
- **InstrumentHeader**: 楽器ヘッダー（学習ツールメニュー含む）

### サービス・ユーティリティ
- **認証サービス**: Google OAuth、メール認証
- **オフラインストレージ**: オフライン時のデータ保存
- **通知サービス**: プッシュ通知、リマインダー
- **音声認識サービス**: Whisper API連携
- **アナリティクス**: イベント追跡
- **エラーハンドリング**: エラー制限機能

### データ管理
- **練習記録**: practice_sessions テーブル
- **目標**: goals テーブル
- **録音**: recordings テーブル
- **楽器**: instruments テーブル
- **組織**: organizations テーブル
- **イベント**: events テーブル
- **スケジュール**: practice_schedules テーブル

## 📊 機能の状態まとめ

- ✅ **有効**: 完全に実装され、使用可能
- 🚧 **準備中**: 実装中だが未完成、一部機能のみ動作
- 💤 **非表示**: 実装済みだがUIから非表示
- ⚠️ **一部非表示**: 機能は実装済みだが一部UI要素が非表示

## 📝 備考

- タブバーに表示されるのは6つのメインタブのみ
- その他の画面は設定画面やヘッダーメニューから遷移
- 組織機能はソロモードとグループモードで表示が異なる
- 一部機能はペイウォール対応（未購読時は制限あり）
